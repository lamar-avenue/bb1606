КАК ЗАГРУЗИТЬ НА GITHUB PAGES

1. Создай новый репозиторий на GitHub, например: bb-1606
2. Загрузи в него файлы:
   - index.html
   - operator_1606.png
3. Открой Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Нажми Save
7. Дождись ссылки.

Итоговая ссылка будет примерно такая:
https://ТВОЙ_НИК.github.io/bb-1606/
