BB_1606: загрузка на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в репозиторий файлы index.html и operator_1606.png.
3. Открой Settings -> Pages.
4. В Source выбери Deploy from branch.
5. В Branch выбери main, папку /root.
6. Сохрани настройки и дождись публикации.
7. Итоговая ссылка будет похожа на:
   https://USERNAME.github.io/bb-1606/

Сайт полностью статический. Сервер, npm, сборщики и внешние библиотеки не нужны.
Можно открыть index.html прямо в браузере, и он будет работать локально.
