birthday signal // blackbox recovery // final handoff

Олег, с днём рождения.

Я попытался сделать что-то на подобии Цикады 3310, но без лишней боли. Здесь будет короткий цифровой квест: видео, музыка, простой шифр, немного Go-вайба и финальный оператор.

Компилятор Go не нужен. Всё можно пройти прямо на этой странице. Подсказки открываются только если ты сам нажмёшь.

Начать

01 // что происходит

Цель:
восстановить финальный ключ доступа.
Финал:
QR-код приведёт к оператору, который вручит подарок.

Ключ состоит из трёх фрагментов. Формат финального ключа:

fragment1-fragment2-fragment3

К медиа-следам

02 // медиа-следы

Не нужно смотреть всё целиком. Ищи смысл в названиях. Один видеоролик подсказывает метод шифрования, один трек подсказывает финальный контакт.

Видео
  1. Cicada 3301: An Internet Mystery
  2. Caesar Cipher
  3. UVB-76 / The Buzzer
  4. Blackbox Life Recorder 21f
Музыка
  1. LEMMiNO — Cipher
  2. Aphex Twin — Blackbox Life Recorder 21f
  3. Jim Croce — Operator
  4. Kraftwerk — The Robots
В видео нужен ролик, название которого прямо говорит, какой шифр использовать.
В музыке нужен трек, название которого связано с человеком, к которому нужно обратиться в финале.
К шифру

03 // Go trace

Это не нужно компилировать. Это просто Go-след, который показывает параметры.

package main

const birthday = "16.06"
const revision = 22

var payload = "JVEKQIRX 1/3: WMKREP"

func main() {
    // not every answer needs a compiler
    // sometimes source is enough
}

Используй метод, который нашёл в видео, и число revision.

revision = 22. Значит сдвиг 22.
После декода должен появиться первый фрагмент: signal.
К следующему блоку

04 // закодированный блок

Один фрагмент сохранён в другом виде. Метод указан в списке методов.

Это Base64. После декода появится второй фрагмент: restored.
Собрать ключ

05 // финальный ключ

Третий фрагмент даёт музыка. Нужен трек, связанный с финальным контактом. Ответ одним словом на английском, маленькими буквами.

Собери ключ:

fragment1-fragment2-fragment3

Фрагменты идут так: шифр → Base64 → музыка.
Ключ выглядит так: signal-restored-operator.

06 // доступ подтверждён

Чёрный ящик восстановлен. Финальный оператор найден.

Отправь оператору фразу доступа:

BB-1606-22
